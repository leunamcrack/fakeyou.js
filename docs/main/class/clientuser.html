# ClientUser

The user logged in with the client

# Constructor

new ClientUser(client, data);
PARAMETER TYPE DESCRIPTION OPTIONAL
client Client The client belongs to No
data APIData (opens new window) The user data logged in No

# Properties

# categories

All of categories the client is currently handling mapped by their ids

  • Type: VoiceManager

# models

All of models the client is currently handling mapped by their ids

  • Type: ModelManager

# isReady

Checks if the client is ready to work

  • Type: Boolean (opens new window)

# results

All of results the client is currently handling mapped by their ids

  • Type: VoiceManager

# session

The user log in data

  • Type: SessionData

# token

The authorization token for data request

  • Type: AuthToken

# user

The user logged in

  • Type: ClientUser or null (opens new window)

# Methods

# fetchUser(query)

Fetch all data about an user

PARAMETER TYPE DESCRIPTION OPTIONAL
query UserResolvable Object or username profile No
  • Return: Promise (opens new window) <Result>

# leaderboard()

Obtains the TTS/W2l leaderboard

  • Return: Promise (opens new window) <LeaderBoard>

# makeRequest(model, text)

Generate a TTS request

PARAMETER TYPE DESCRIPTION OPTIONAL
model ModelResolvable Name, object or token model No
text String (opens new window) The text to speech No
  • Return: Promise (opens new window) <Result>

# searchModel(query)

Search an model

PARAMETER TYPE DESCRIPTION OPTIONAL
query ModelResolvable Name, object or token model No
  • Return: Group <ModelToken, Model>

# start()

Fetch all data and login if needed

  • Return: Promise (opens new window) <Client>
