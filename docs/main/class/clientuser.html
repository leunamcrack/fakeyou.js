# User

The user logged in into the client

# Constructor

new ClientUser(client, data);
PARAMETER TYPE DESCRIPTION OPTIONAL
client Client The client that instantiated this No
data APIData (opens new window) The category data No

# Properties

# badges

The badges for this user

  • Return: Badges or null (opens new window)

# createdAt

The date the user was created

  • Return: Date (opens new window) or null (opens new window)

# createdTimestamp

The timestamp in milliseconds the user was created

  • Return: Number (opens new window) or null (opens new window)

# description

The user description (markdown style)

  • Return: String (opens new window) or null (opens new window)

# displayName

The user display name

  • Return: String (opens new window)

# gravatarHash

The Gravatar (opens new window) hash

  • Return: String (opens new window)

# isBanned

Checks if the client user is banned from FakeYou

  • Return: Boolean (opens new window)

# isMod

Checkis if the client user is a FakeYou moderator

  • Return: Boolean (opens new window)

# links

The social links this belongs to

  • Return: LinksData

# partial

Checks if has partial data

  • Return: Boolean (opens new window)

# plan

The client user's FakeYou plan type

  • Return: String (opens new window)

# token

The user token

  • Return: UserToken

# ttsVisibility

The default TTS visibility of this user

  • Return: String (opens new window) or null (opens new window)

# username

The username of the user

  • Return: String (opens new window)

# w2lVisibility

The default W2L visibility of this user

  • Return: String (opens new window) or null (opens new window)

# Methods

# edit(options)

Edit the client user info

PARAMETER TYPE DESCRIPTION OPTIONAL
options ClientEditOptions The client user options to edit No
  • Return: Promise (opens new window) <ClientUser>

# fetch()

Get all data about this user

  • Return: Promise (opens new window) <ClientUser>

# gravatarURL()

Obtains the URL of gravatar image

  • Return: String (opens new window)

# profileURL()

Obtains the URL of FakeYou profile

  • Return: String (opens new window)

# ttsModels()

Get the models made by the client user

  • Return: Promise (opens new window) <Group <ModelToken, Model>>

# ttsResults()

Get the TTS results generated by the client user

  • Return: Promise (opens new window) <Group <ResultToken, TTSResult>>
