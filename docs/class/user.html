# User

Represents a user on FakeYou

# Constructor

new User(client, data);
PARAMETER TYPE DESCRIPTION OPTIONAL
client Client The client that instantiated this No
data APIData (opens new window) The category data No

# Properties

# badges

The badges for this user

  • Return: Badges or null (opens new window)

# createdAt

The date the user was created

  • Return: Date (opens new window) or null (opens new window)

# createdTimestamp

The timestamp in seconds the user was created

  • Return: Number (opens new window) or null (opens new window)

# description

The user description (markdown style)

  • Return: String (opens new window) or null (opens new window)

# displayName

The user display name

  • Return: String (opens new window)

# gravatarHash

The Gravatar (opens new window) hash

  • Return: String (opens new window)

# links

The social links this belongs to

  • Return: LinksData

# partial

Checks if has partial data

  • Return: Boolean (opens new window)

# token

The user token

  • Return: UserToken

# ttsVisibility

The default TTS results visibility of the user

  • Return: Visibility or null (opens new window)

# username

The username of the user

  • Return: String (opens new window)

# w2lVisibility

The default W2L results visibility of the user

  • Return: Visibility or null (opens new window)

# Methods

# fetch()

Get all data about this user

  • Return: Promise (opens new window) <User>

# gravatarURL()

Obtains the URL of gravatar image

  • Return: String (opens new window)

# isVisible(isw2l)

Checks if the category is visible to public

PARAMETER TYPE DESCRIPTION OPTIONAL
isw2l Boolean (opens new window) If you want to get w2l option Yes
  • Return: Boolean (opens new window) or null (opens new window)

# profileURL()

Obtains the URL of FakeYou profile

  • Return: String (opens new window)
