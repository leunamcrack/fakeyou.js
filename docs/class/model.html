# Model

Represents a FakeYou TTS model

# Constructor

new Model(client, data);
PARAMETER TYPE DESCRIPTION OPTIONAL
client Client The client that instantiated this No
data APIData (opens new window) The model API data No

# Properties

# algorithm

The algorithm type used in the model

  • Type: String (opens new window) or null (opens new window)

# category

The category group in which the model is located

  • Type: Group <CategoryToken, Category> or null (opens new window)

# categoryTokens

The list of category tokens in which the model is located

  • Type: Array (opens new window) <CategoryToken> or null (opens new window)

# client

The client that instantiated this

  • Type: Client

# count

The count of generated requests by the model

  • Type: Number (opens new window) or null (opens new window)

# createdAt

The date this model was created

  • Type: Date (opens new window) or null (opens new window)

# createdTimestamp

The time in seconds this model was created

  • Type: Number (opens new window) or null (opens new window)

# description

The model description (markdown style)

  • Type: String (opens new window) or null (opens new window)

# features

The model featured info

  • Type: ModelFeatures

# lang

The ISO 639-1 (opens new window) language code

  • Type: String (opens new window)

# locate

The IEFT BCP 47 (opens new window) language tag

  • Type: String (opens new window)

# name

A maybe suggested name for use in apps

  • Type: String (opens new window) or null (opens new window)

# partial

Checks if has partial data

  • Type: Boolean (opens new window)

# pipeline

The model text pipeline type

  • Type: String (opens new window) or null (opens new window)

# setVisibility

The visibility of the model

  • Type: Visibility or null (opens new window)

# title

The title of the model

  • Type: String (opens new window)

# token

The model token

  • Type: ModelToken

# type

The type of the model

  • Type: String (opens new window)

# updatedAt

The date this model was updated

  • Type: Date (opens new window) or null (opens new window)

# updatedTimestamp

The time in seconds this model was updated

  • Type: Number (opens new window) or null (opens new window)

# user

The user who created the model

  • Type: User or null (opens new window)

# userToken

The user token who created the model

  • Type: UserToken or null (opens new window)

# vocoder

The model's default pretrained vocoder

  • Type: String (opens new window) or null (opens new window)

# Methods

# fetch()

Get all data about this model

  • Return: Promise (opens new window) <Model>

# isVisible()

Checks if the category is visible to public

  • Return: Boolean (opens new window) or null (opens new window)

# modelURL()

Obtains the URL of FakeYou model page

  • Return: String (opens new window)

# request(text)

Make a TTS request

PARAMETER TYPE DESCRIPTION OPTIONAL
text String (opens new window) The text to speech No
  • Return: Promise (opens new window) <TTSResult>
