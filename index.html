A powerful library for interacting with the FakeYouAPI easily

npm install fakeyou.js

# About

fakeyou.js is a Node.js (opens new window) module that allows you to interact with the FakeYouAPI (opens new window) easily. Adopts an object-oriented approach that makes your code much neater and easier to understand.

It can be used in discord.js (opens new window) or another proyect where you need it, also has the most possible coverage of the API and now this package is recognized by FakeYou (opens new window)

# Example

const FakeYou = require('fakeyou.js');
const fy = new FakeYou.Client({
    token: 'anOptionalSecretToken',
    usernameOrEmail: 'anOptionalCoolUsernameOrEmail',
    password: 'anOptionalSuperSecretPassword'
});
await fy.start(); //required

let models = fy.searchModel('mario');
if(models.size >= 1) {
  let result = await fy.makeTTS(models.first(), 'A cool text to speech');
  result.audioURL();
}
