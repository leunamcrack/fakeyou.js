# About

FakeYou.JS is a Node.js (opens new window) module that allows you to interact with the FakeYou API (opens new window) easily.

It can be used in discord.js (opens new window) or another proyect where you need it, also has nearly 80% coverage of the API. This package is not afiliate with fakeyou.com (opens new window)

# Example:

const FakeYou = require('fakeyou.js');
const fy = new FakeYou.Client('optional token'); //Check more information about this in docs.fakeyou.com//

fy.on('ready', async () => {
  await fy.login({
    usernameOrEmail: 'acoolusername',
    password: 'asupersecretpass'
  });
  let model = fy.searchModel('mario');
  if(model) {
    await fy.makeVoice(model, "I'll only say the things you want me to say, and nothing more");
  };
})
